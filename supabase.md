# ⚡ Complete Supabase Setup Guide for AgriSaathi AI

This guide replaces the Firebase setup. Supabase provides Open Source Auth, Database, and Storage.

---

## 🏗️ Step 1: Create Your Supabase Project
1. Go to [database.new](https://database.new) (Supabase Dashboard).
2. Click **"New Project"**.
3. Name it: `AgriSaathi AI`.
4. **Database Password**: Set it and **SAVE IT** somewhere safe.
5. **Region**: Choose **Mumbai (ap-south-1)** for best speed in India.
6. Click **"Create new project"**.

---

## 🔑 Step 2: Get Your API Keys
1. Once the project is ready, go to **Project Settings** (gear icon) > **API**.
2. Copy your **Project URL** (e.g., `https://xyz.supabase.co`).
3. Copy your **anon public API Key**.
4. Create a `.env.local` file in your root folder and paste them:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_project_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
   ```

---

## 🔐 Step 3: Enable Authentication
Go to **Authentication > Providers**:

### 1. Email (OTP/Password)
- Enabled by default.

### 2. Google
- Click **Google** and toggle **Enable**.
- You will need a **Client ID** and **Secret** from the [Google Cloud Console](https://console.cloud.google.com/).
- Copy the **Redirect URI** from Supabase and paste it into your Google Cloud project's "Authorized redirect URIs".

### 3. Facebook
- Click **Facebook** and toggle **Enable**.
- Get your **App ID** and **Secret** from [Meta for Developers](https://developers.facebook.com/).
- Copy the **Callback URL** from Supabase and paste it into your Facebook Login settings.

---

## 📂 Step 4: Database Setup (SQL)
Supabase handles data in tables. Go to **SQL Editor** and run this to create your basic tables:

```sql
-- Create a users profile table
CREATE TABLE profiles (
  id UUID REFERENCES auth.users ON DELETE CASCADE,
  full_name TEXT,
  phone_number TEXT,
  location TEXT,
  updated_at TIMESTAMP WITH TIME ZONE,
  PRIMARY KEY (id)
);

-- Create a chat logs table
CREATE TABLE chat_logs (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  user_id UUID REFERENCES auth.users,
  message TEXT,
  role TEXT, -- 'user' or 'ai'
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```
